# Challenge-Corner
Challenge Corner edisi 20 ini bertujuan mengajak pembaca Pelindo Corner untuk menyaksikan video CEO Direction yang inspiratif, lalu merangkum key message versi masing-masing. Melalui pengalaman interaktif ini, setiap peserta tidak hanya menonton, tetapi juga merefleksikan arah, semangat, dan pesan kepemimpinan Pelindo.
